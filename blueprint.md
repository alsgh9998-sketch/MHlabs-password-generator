# Password Generator

## Overview

This project is a web-based password generator that allows users to create strong, random passwords. The application is a single HTML file with inline CSS and JavaScript. The user can specify the password length and choose to include uppercase letters, lowercase letters, numbers, and symbols.

## Implemented Features

*   **UI:**
    *   A simple and clean user interface.
    *   An input field to specify the password length.
    *   Checkboxes to include/exclude character types.
    *   A button to generate a new password.
    *   A display area for the generated password.
    *   A button to copy the password to the clipboard.

*   **Functionality:**
    *   Generates a random password based on the user-selected criteria.
    *   Copies the generated password to the clipboard.
