
# Blueprint: AdSense Optimized Password Generator Site

## 1. Project Overview

This project is a web-based password generator tool. The primary goal is to provide a user-friendly and secure way for users to create strong, random passwords. To ensure a high chance of Google AdSense approval, the site will be enhanced with high-quality, informative content, clear navigation, and essential trust-building pages.

## 2. Design and Features (Current State)

### Implemented Features:

*   **Password Generation:**
    *   Generates random passwords based on user-defined criteria.
    *   Options to include uppercase letters, lowercase letters, numbers, and symbols.
    *   Adjustable password length.
*   **User Interface:**
    *   Clean and modern design.
    *   Dark/Light theme switcher.
    *   Language selection (English/Korean).
    *   Responsive design for mobile and desktop.
*   **Core Technologies:**
    *   HTML, CSS, JavaScript (no frameworks).
*   **Analytics:**
    *   Google Analytics (gtag.js) implemented across all pages for traffic analysis.
